# 铁炉咖啡线上网站

以铁炉咖啡真实门店照片、官方 Logo 与公开品牌资料为基础的单页网站。视觉使用暖米白纸张、木色与克制的炉火红，围绕“进来坐坐”营造小众、安静、有人情味的咖啡空间。

在线访问：<https://zhangliyuan070923-create.github.io/tielu-coffee/>

## 本地运行

```bash
npm install
npm run dev
```

浏览器打开 <http://127.0.0.1:5173/>。

正式构建：

```bash
npm run build
```

## 页面内容

- “进来坐坐”真实门店首屏
- 炉边日常时间线
- 品牌手记与横向照片浏览
- 可交互的城市温度滑杆
- 河下店、海创空间店两家已确认门店
- 官方抖音入口
- 桌面端与移动端响应式布局

图片与资料出处见 [来源清单.md](./来源清单.md)。未确认的地址、营业时间、菜单与价格没有擅自补写。
